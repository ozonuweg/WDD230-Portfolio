Reading on Local Storage: Remembering Users Info

  • Information stored or shared with the User is called OUTPUT.
  • Anything coming in from where it was stored or anything the user enters is called INPUT.
  • Input & Output are called IO, Input-Output.
  • In HTML, when you want to get user's input or text area tag, use the word VALUE.
  • In HTML, When you want to show something to a user (section), use innerHTML.
